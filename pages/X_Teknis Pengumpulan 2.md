# Teknis Pengumpulan

<div>

Format setiap file `.java` didahulukan dengan Nama, NPM, Kelas, Tanggal, dan Deskripsi

</div>

Cara menambah comment di java

```java {all}
// untuk single line
/* untuk multiple line */
```

Contoh Format

```java {all}
/*
  Nama	: Jane Doe
  NPM		: 99
  Kelas		: A
  Tanggal	: 1 September 2021
  Deskripsi	: Class jawaban exercise-01 soal-01
*/

```
